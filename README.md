## 搬瓦工修改机房是什么：先说清楚这个功能本身

搬瓦工的 KiwiVM 控制面板里有一个叫 **Migrate to another DC** 的选项。DC 就是 Data Center，数据中心的意思。点进去选好目标机房，点确认，系统会自动把你 VPS 里的文件和配置原封不动地搬过去，IP 地址会变成新机房的 IP，其他数据不丢。

迁移本身是免费的，不收费，也没有官方规定的次数上限。整个过程通常 10 到 20 分钟，文件多的情况下可能要久一点，期间 VPS 最后几分钟会短暂关机，完成后会发邮件通知你。

## 操作步骤：怎么修改搬瓦工机房

操作流程本身不复杂，按这几步走：

1. 登录搬瓦工官网，进入 **Services → My Services**，找到你的 VPS 实例
2. 点击该实例右侧的 **KiwiVM Control Panel** 按钮进入后台控制面板
3. 在左侧菜单找到 **Migrate to another DC**，点进去
4. 右侧会列出当前套餐可切换的机房列表，选好目标机房，点 **Confirm on next step**
5. 确认界面再次出现后，点 **Start Migration** 开始迁移
6. 等待 10~20 分钟。迁移完成后会收到搬瓦工发来的邮件通知，邮件里附有新机房的 IP 地址

完成之后，用新 IP 重新 SSH 连接即可。如果你的 VPS 上跑着网站，记得同步更新域名解析，把 A 记录改成新 IP，等 DNS 生效就恢复了。

顺便说一句：如果你从 DC2 切到 DC4，再切回 DC2，理论上会回到原来那个 IP（前提是这个 IP 没被别人占走）。不想换 IP 只想测试不同机房线路质量的话，可以试试这个方式。

## 哪些套餐支持修改机房？这点最容易踩坑

不是买了搬瓦工就能随便换机房的。套餐决定了你能切换哪些机房，甚至能不能切换。

**普通 KVM 套餐**：只能在普通机房之间迁移，可选范围包括 DC2 QNET、DC4 MCOM、DC8 ZNET、弗里蒙特（FMT）、新泽西（USNJ）、荷兰（EUNL\_3）等。切到 CN2 机房（DC3/DC8）后流量会按 0.33 倍计算。

**CN2 GIA-E 套餐（洛杉矶电商套餐）**：这档套餐是目前支持迁移机房最多的，可切换的节点包括 DC6 CN2 GIA-E、DC9 CN2 GIA、日本软银（JPOS\_1）、荷兰联通（EUNL\_9）、加拿大温哥华等，多达十几个机房。买这档的人通常就是冲着这个灵活性来的。

**香港 CN2 GIA 套餐**：高端套餐，延迟极低，但历史上有部分款式是单机房锁定的，无法迁移。购买前要看清楚套餐描述里有没有 "Location: Hong Kong Only" 之类的限制说明。

**限量版套餐（Box 系列、THE PLAN 等）**：迁移范围因款式而异。THE PLAN 系列支持十几个机房自由切换，算是最灵活的那档；MINICHICKEN 之类的入门限量版则通常只支持单机房，买前要确认。

有个简单的判断方法：进购买页面，看套餐描述里有没有列出"可选机房"的清单。如果只写了一个固定机房名，基本就是不可迁移的。

## 最该知道的一个坑：IP 被封之后不能修改机房

这点坑了不少人。

很多人第一反应是：IP 被封了，换个机房不就换了 IP 吗？逻辑没错，但搬瓦工的官方政策就是——IP 被封之后，Migrate to another DC 这个功能会被自动锁住，操作界面会反复出现"Please try again in 10-15 minutes"的提示，不管等多久都一样。

这是搬瓦工为了防止有人把"换机房"当成无限免费换 IP 的工具而设置的限制。功能设计的本意是让你优化线路选择，不是绕过 IP 限制的后门。

遇到这种情况只有一条路：付费更换 IP，目前价格在 $7.39 到 $8.79 之间（以官网结算页面实时价格为准）。IP 换好之后，切换机房的功能就恢复了。

怎么判断 IP 是真的被封了？进 KiwiVM 后台，把 URL 最后的 `main.php` 替换成 `main-exec.php?mode=blacklistcheck`，点 Test Main IP，出现 IP BLOCKED 就是封了，出现 IP NOT BLOCKED 说明 IP 本身没问题，可能是别的原因连不上。

## 常见问题 FAQ

**Q：搬瓦工修改机房会丢失 VPS 里的数据吗？**

A：正常迁移不会丢。迁移过程会把整个 VPS 的文件和配置原样搬到新机房，官方的机制就是这样设计的。不过在执行任何涉及底层变动的操作前，养成先做快照备份的习惯是最安全的。

**Q：切换机房完全免费吗？**

A：对，迁移机房本身不收费，次数也没有硬性限制。前提是 IP 没被封，且你的套餐支持目标机房的迁移。

**Q：迁移完成之后 IP 一定会变吗？**

A：换到不同机房会换 IP。但有一个特殊情况：从 A 机房切换到 B，再切回 A，通常会找回原来的 IP（除非这个 IP 已经被别人占走）。

**Q：机房列表里没有我想要的机房怎么办？**

A：这说明你当前套餐不支持迁移到那个机房。想切换更多机房的选项，只能升级到支持对应机房的套餐。CN2 GIA-E 套餐支持的节点最多，如果想要更大的选择空间可以考虑升级。

**Q：迁移的时候需要关机吗？**

A：不需要手动关机。迁移过程中 VPS 大部分时间都可以正常访问，只有最后阶段会短暂关闭几秒到几分钟，系统会自动处理。

**Q：新购买时如果选错了机房，可以换吗？**

A：可以，买对了套餐的话，进 KiwiVM 后台按上面的步骤操作就行，不需要退款重买。
